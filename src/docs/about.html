Tabby Tab

For each configuration, make sure that the Original List has a return where ever you would like a tab placed.

For instance:
FirstName[RETURN]
LasTName[RETURN]
Title1[RETURN]
Title2[RETURN]

Will produce:
FirstName [TAB] LastName [TAB] Title1 [TAB] Title2

Make sure that you're doing a badge setup of only one type (ie. only 4 tabs, or only 5 tabs), as you will get an error if you attempt to tab out a 5 tab list and one of the names only has four lines.

Name List Editor 1.0

The Name List Editor was designed to scan an imported name list for multiple spaces between first name & last name, as well as to scan for either trailing or leading spaces on the names. In any instance where a leading, a trailing, or multiple spaces were found, those spaces would be either replaced with a single space or removed in the case of a leading or trailing space.
